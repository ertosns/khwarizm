What is XML?

Extensible Markup Language (XML) is general purpose data formate. Designed in 1996 to transfer data between systems or applications.

Why XML is used instead of formal english language?

because it defined by document type definition (DTD) as an agreement between programmers.

What XML use for nowadays?

  • interfacing with SOAP system
  • building with ant
  • creating android layouts
  • passing messages between network systems
  • RSS

What are other data format systems or replacements to XML?

  • jason
  • yaml
